Selected work

Six projects.

01 / 06

Parlance

A visual workspace for orchestrating multi-agent Claude AI workflows — no code, no YAML, just a node-link canvas. Connect AI agents, file sources, and output endpoints with edges, then run them with real API calls. The execution engine handles parallel wave execution, three-tier retry escalation, and targeted Overseer feedback that re-runs only the agent responsible — not the entire workflow. BYOK, fully browser-based.

React 19 Multi-agent Orchestration ReactFlow Vercel serverless Open Source BYOK
Live app GitHub
02 / 06

Claude Batch

A free, open-source tool that queues prompts to Claude and fires them at a controlled, user-defined pace — so you never hit a rate limit mid-workflow. Built for repetitive batch work: data analysis, document processing, running the same prompt across many inputs. Runs entirely in the browser, bring-your-own-key, nothing ever touches a server.

Claude API Queue management Open Source Client-side BYOK
Live app GitHub
03 / 06

LoopLens

A cost simulator for agentic AI loops — see per-iteration cost breakdowns before running anything. Models fan-out multi-agent architectures, calculates prompt caching break-even, and compares 13+ models side by side on the same loop configuration. Pure deterministic math, zero API calls required to simulate.

React Vite Open Source Cost modeling BYOK
Live app GitHub
04 / 06

Singularity Workspace

A free, open-source visual thinking platform — node canvas, kanban, timeline, table view, and an AI assistant, all in one. Built as a real alternative to Monday.com, Notion, and Miro. No subscriptions, no accounts, no data leaving your machine.

React Vite Open Source AI assistant BYOK
Live app GitHub
05 / 06

Reality Check

An AI-powered fact-checking tool that evaluates headlines and claims across four dimensions: source credibility, evidence quality, context, and emotional framing — with live web research via Claude. Fully client-side. Your API key never leaves your browser.

React Tailwind Claude API Web search Media literacy
Live app GitHub

Intelligent Trader IT/V4

A three-stage autonomous algorithmic trading system built on statistical anomaly detection. Six walk-forward runs. The core engine uses z-score–based deviation analysis, a custom capital allocation model (MSIC), and a portfolio heat limiter capped at 10% concurrent open risk.
